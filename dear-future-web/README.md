# Dear Future — Web Arayüzü

Dear Future platformunun kullanıcıya yönelik web arayüzü. React, Vite, GSAP, Lenis ve Motion ile geliştirilmiştir.

## Gereksinimler

- Node.js 18+
- npm veya yarn

## Kurulum

```bash
npm install
```

## Geliştirme Sunucusu

```bash
npm run dev
```

Tarayıcıda Vite’ın gösterdiği adrese (genelde `http://localhost:5173`) gidin.

## Build

```bash
npm run build
```

Üretim çıktısı `dist` klasörüne alınır.

## Lint

```bash
npm run lint
```
