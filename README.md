# Dear Future

**Dear Future**, geleceğe yazılan mektupları güvenle saklamanızı ve belirlediğiniz tarihte ulaşmanızı sağlayan bir platformdur.

🌐 **Uygulama adresi:** [dearfuture.com.tr](https://dearfuture.com.tr)

Siteden uygulamayı kullanabilir, **Android APK dosyasını indirebilir** ve mobil cihazınızda Dear Future’ı kurabilirsiniz.

---

## Proje Yapısı

Bu depo aşağıdaki projeleri içerir:

| Proje | Açıklama |
|-------|----------|
| [DearFuture](./DearFuture) | Backend API (Spring Boot) |
| [dear-future-web](./dear-future-web) | Kullanıcı web arayüzü (React + Vite) |
| [dear-future-admin](./dear-future-admin) | Yönetim paneli (React + Vite) |
| [dear_future_mobile](./dear_future_mobile) | Mobil uygulama (Flutter) |

---

## Hızlı Başlangıç

1. **Backend:** `DearFuture` klasöründe `./mvnw spring-boot:run`
2. **Web:** `dear-future-web` içinde `npm install` ve `npm run dev`
3. **Admin:** `dear-future-admin` içinde `npm install` ve `npm run dev`
4. **Mobil:** `dear_future_mobile` içinde `flutter pub get` ve `flutter run`

Detaylı kurulum için her projenin kendi `README.md` dosyasına bakın.
